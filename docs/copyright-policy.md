# Copyright Policy

This project is designed to be public and reusable. Copyright safety is therefore a core editorial requirement.

## Do Not Commit

- Copyrighted PDFs, EPUBs, or scans.
- OCR dumps from copyrighted books.
- Full chapters or long passages from copyrighted sources.
- Long copied practice instruction blocks.
- Publisher images unless the license permits reuse.

## Allowed Public Data

- Bibliographic metadata.
- Short citations where appropriate.
- Source identifiers.
- Original summaries.
- Original encyclopedia prose.
- Public-domain source excerpts.
- Links to external resources.
- License-compatible datasets.

## Local Private Corpus

Contributors may use private local corpora for research if they have lawful access, but the public repository should only contain the original article prose, structured metadata, and source references.

## Quotation Discipline

Use quotations sparingly. Prefer paraphrase and citation. When a short quotation is necessary, keep it brief and attach source metadata.

## Takedown

If you believe material in this repository infringes copyright, open an issue or contact a maintainer with the file path and reason. Maintainers should remove or quarantine disputed material while reviewing it.
