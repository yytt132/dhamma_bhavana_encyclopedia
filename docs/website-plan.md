# Website Plan

The public website should make the encyclopedia useful to non-technical Korean-speaking practitioners.

## v0.2 Prototype

- Static landing page.
- Entry renderer.
- Glossary renderer.
- Comparison renderer.
- Review-status badges.
- Source metadata panel.
- Related-entry navigation.
- Search index.

## Design Principles

- Reading first.
- Korean-language accessibility first.
- Clear distinction between reviewed and draft material.
- Source metadata visible but not overwhelming.
- No copyrighted source text.

## Future Information Architecture

- Encyclopedia essays.
- Glossary.
- Practice maps.
- Comparisons.
- Traditions.
- Sources.
- Contributor guide.
