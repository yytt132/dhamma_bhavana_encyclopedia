# Support

Use GitHub Issues for project support.

간단한 오류 제안, 오탈자, 참고자료 추천은 이메일로도 보낼 수 있습니다.

- Email: pnymeditation@gmail.com

## Good Issue Types

- Entry proposal.
- Source metadata correction.
- Copyright concern.
- Editorial review request.
- Website/tooling bug.
- Schema improvement.

## Suggested Formats

### 오류·오탈자 제보

- 위치:
- 현재 내용:
- 제안:

### 참고자료 추천

- 자료명:

- 항목명:
- 원어 표기:
- 분류:
- 왜 필요한가:
- 출처 후보:

## Before Opening an Issue

Please check:

- `README.md`
- `CONTRIBUTING.md`
- `docs/editorial-guide.md`
- `docs/copyright-policy.md`
- existing issues

## Copyright Concerns

If you believe material in the repository should not be public, open an issue with:

- file path
- reason
- source or rights concern

Maintainers should remove or quarantine disputed material while reviewing it.
