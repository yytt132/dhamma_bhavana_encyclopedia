한국어 수행자를 위한 공개 불교·명상 수행 백과 프로젝트입니다.

**Dhamma Bhavana Encyclopedia**는 불교와 명상·수행 전통을 출처 기반으로 정리하는 오픈소스 백과입니다. 최종 목표는 한국어를 사용하는 수행자, 명상지도사, 불교 공부를 시작하는 사람, 번역자, 연구자들이 함께 사용할 수 있는 신뢰 가능한 수행 지식 기반을 만드는 것입니다.

## 왜 이 프로젝트를 만드는가

영어권에는 불교와 수행을 공부할 수 있는 좋은 자료 생태계가 이미 많습니다. Stanford Encyclopedia of Philosophy, Digital Dictionary of Buddhism, Princeton Dictionary of Buddhism, SuttaCentral, 84000 같은 자료와 플랫폼이 있습니다.

하지만 한국어권에는 수행자와 명상지도사, 불교 공부를 시작하는 사람, 깊이 있는 수행론을 이해하려는 사람이 쉽게 접근할 수 있는 공개적이고 신뢰할 만한 수행 백과가 충분하지 않습니다.

불교 사전과 대사전은 존재합니다. 그러나 실제 수행 개념을 공부하고 지도하는 데 필요한 기초적이고 체계적인 자료, 특히 원어와 번역어, 전통별 의미 차이, 수행 맥락, 현대적 오해, 참고문헌을 함께 정리한 공개 자료는 아직 부족합니다.

이 프로젝트는 그 빈자리를 채우기 위해 시작되었습니다.

## 누구를 위한 백과인가

- 한국어로 불교 수행을 공부하는 수행자
- 명상지도사와 불교명상지도사
- 불교 공부를 시작하는 입문자
- 수행 개념을 더 깊게 이해하려는 중급 학습자
- 불교 원전과 수행 문헌을 번역하는 번역자
- 한국불교, 남방불교, 티벳불교를 비교하며 공부하는 사람
- 디지털 불교학과 오픈소스 지식 기반에 관심 있는 기여자

## 어떤 백과를 만들 것인가

이 프로젝트는 단순한 용어풀이 사전이 아닙니다. 각 개념을 가능하면 다음 요소와 함께 설명합니다.

- 원어와 번역어
- 전통별 의미 차이
- 문헌적 배경
- 철학적 의미
- 실제 수행에서의 역할
- 유사 개념과의 차이
- 현대적 사용과 오해
- 관련 원전과 참고문헌
- 관련 항목과 비교 항목

예를 들어 `bhavana`, `sati`, `samatha`, `vipassana`, `samadhi`, `dhyana/jhana`, `metta`, `sunyata`, `zazen`, `pranayama` 같은 개념은 같은 번역어로 단순히 묶기 어렵습니다. 전통과 문헌, 수행 목적에 따라 의미가 달라지기 때문입니다.

따라서 이 백과는 같은 표면어를 하나로 합치기보다, 출처와 전통을 보존하면서 조심스럽게 비교하는 방향을 지향합니다.

## 프로젝트의 기본 원칙

- **한국어 우선**: 한국어 수행자가 읽고 사용할 수 있는 설명을 우선합니다.
- **출처 기반**: 주장과 설명은 가능한 한 출처, 문헌, 전통 맥락과 함께 기록합니다.
- **전통 구분**: 한국불교, 남방불교, 티벳불교, 현대 명상, 요가·기공·기독교 관상 등 서로 다른 맥락을 함부로 합치지 않습니다.
- **수행자 중심**: 학술적 정확성을 추구하되 실제 수행자와 지도자가 이해할 수 있는 언어를 지향합니다.
- **저작권 안전**: 저작권이 있는 PDF, OCR 전문, 책의 긴 발췌문, 수행 지침 전문은 공개 저장소에 올리지 않습니다.
- **인간 검토**: AI를 연구 보조와 초안 작성에 활용할 수 있지만, 최종 해석과 공개 상태 변경은 인간 검토를 거칩니다.

## 창립자와 메인테이너

창립자이자 현재 메인테이너는 한국에서 활동하는 불교명상지도사이자 명상지도사입니다. 약 10년 정도 명상 수행을 이어왔으며, 한국불교, 남방불교, 티벳불교를 모두 공부하고 있습니다. 또한 신수대장경, 빨리대장경, 티벳대장경을 한국어로 번역하는 작업도 진행하고 있습니다.

이 배경은 프로젝트의 방향과 직접 연결됩니다. 이 백과는 추상적인 데이터 프로젝트가 아니라, 한국어 수행자와 지도자, 번역자, 공부하는 사람들이 실제로 필요로 하는 자료를 공개적으로 정리하려는 시도입니다.

## 공개될 공간

완성된 백과는 기존 Dhamma Reader 생태계에 통합해 공개할 예정입니다.

- 공개 사이트: [dhammareader.com](https://dhammareader.com/)

이 저장소는 백과 항목, 용어 항목, 비교 글, 온톨로지 스키마, 편집 기준, 검증 도구를 관리하는 원천 저장소입니다. Dhamma Reader는 검토된 항목을 실제 사용자가 읽을 수 있는 공개 공간으로 제공하는 역할을 하게 됩니다.

## 저장소 구조

```text
apps/web/                 정적 웹 프로토타입
content/entries/          SEP식 소논문 항목
content/glossary/         짧은 용어 항목
content/comparisons/      비교 항목과 비교 글
content/dossiers/         항목 작성 전 연구 노트
content/templates/        항목 작성 템플릿
data/public/              공개 가능한 메타데이터 샘플
docs/                     사명문, 편집 기준, 로드맵, 정책 문서
packages/schema/          JSON 스키마
scripts/                  검증 및 유지관리 스크립트
```

## 항목 상태

각 항목은 공개 품질을 구분하기 위해 상태를 가집니다.

- `stub`: 뼈대만 있는 상태
- `draft`: 초안
- `sourced`: 출처 메타데이터가 연결된 상태
- `reviewed`: 사람의 검토를 받은 상태
- `published`: 공개 사이트에 안정적으로 올릴 수 있는 상태
- `needs-work`: 수정이 필요한 상태

## English Overview

**Dhamma Bhavana Encyclopedia** is an open-source, source-aware encyclopedia of Buddhist and contemplative practice for Korean-speaking practitioners.

English-speaking students of Buddhism already have access to strong reference ecosystems such as the Stanford Encyclopedia of Philosophy, the Digital Dictionary of Buddhism, the Princeton Dictionary of Buddhism, SuttaCentral, and 84000. Korean-speaking practitioners still lack a comparable open, reliable, practice-oriented encyclopedia.

This project aims to build that missing Korean-language knowledge base. It combines SEP-style essays, Buddhist dictionary-style lexical entries, practice ontology, source metadata, editorial workflow, and future integration with [Dhamma Reader](https://dhammareader.com/).

The project is Korean-first, but English documentation is provided so that international contributors and reviewers can understand the mission and participate.
