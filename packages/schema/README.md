# Schema Package

This directory contains JSON schemas for public data used by the encyclopedia.

The schemas are intended to support:

- Validation before publication.
- Search index generation.
- Related-entry graph construction.
- Editorial review dashboards.
- Future static site rendering.

Current schemas:

- `entry.schema.json`
- `dossier.schema.json`
- `source.schema.json`
- `claim.schema.json`
