# Contributing

Thank you for helping build the Dhamma Bhavana Encyclopedia.

This project welcomes contributions in code, data modeling, bibliography, editing, translation, and subject review.

## Contribution Types

- New encyclopedia entry.
- New glossary entry.
- New comparison essay.
- Source metadata correction.
- Schema improvement.
- Editorial review.
- Website or tooling improvement.

## Entry Workflow

1. Open an issue for the entry or improvement.
2. Create or update a dossier in `content/dossiers`.
3. Draft the article in `content/entries`, `content/glossary`, or `content/comparisons`.
4. Add source metadata rather than copyrighted source text.
5. Set a realistic `status` in front matter.
6. Request review.

## Copyright Rules

Do not add:

- Copyrighted PDFs or EPUBs.
- OCR dumps from copyrighted books.
- Long quotations from copyrighted works.
- Full practice instruction blocks copied from books.
- Scanned images unless they are clearly public domain or compatibly licensed.

You may add:

- Original prose.
- Short quotations where legally appropriate.
- Bibliographic metadata.
- Public-domain source references.
- Links to external public resources.
- Structured summaries written in your own words.

## Style

- Be careful with original languages.
- Do not treat similar English translations as identical concepts.
- Preserve tradition and lineage context.
- Mark uncertainty explicitly.
- Separate doctrine, scholarship, practice instruction, clinical claims, and popular claims.
