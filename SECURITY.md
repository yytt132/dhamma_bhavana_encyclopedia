# Security Policy

This repository primarily contains content, schemas, documentation, and future website code.

## Supported Versions

The `main` branch is the current supported branch.

## Reporting a Vulnerability

Please report security issues by opening a private communication channel with the maintainer when possible. If private contact is unavailable, open an issue with minimal public detail and mark it as security-related.

Do not publicly disclose exploit details before a maintainer has had time to review and respond.

## Content Safety

For this project, security also includes repository safety:

- Do not commit API keys.
- Do not commit private corpus files.
- Do not commit copyrighted PDFs, EPUBs, OCR dumps, or extracted chapters.
- Do not commit personally identifying information.
